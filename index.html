Simply Facebook Insights

is a light weight web app that takes your FB Insight CSV download file and turns it into a interactive dashboard filled with data points and charts to guide you to the health of your FB page.

Instruction

Export a most current Facebook Insight file in CSV format, following the metrics below:

  • Select Data Range: Starts today and ends in 7 to 28 days
    • 7 to 14 days produce the best viewing results
  • Select data type: Page level data
  • Select export type: New
  • Select file format: Comma-spearated values (.csv)
